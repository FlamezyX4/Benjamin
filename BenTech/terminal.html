BenOs Terminal

Welcome to BenOs Terminal - your gateway to exploring this custom OS experience through commands. Although this isn't a real terminal that can connect to server, it's coded to simulate that feel and make your journey into BenOs more cooler.

You can type simple commands below in the future, more responses will be added. For now, try typing some of these:

  • whoami - Returns your username
  • help - Display a list of available commands
  • clear - Clears the terminal screen
  • launch app - Simulates launching a BenOs application
  • echo - Repeats whatever you type after it

This terminal is still under development. it's one of the most fun part of BenOs and a great place to practice javascript interactivity later on.
